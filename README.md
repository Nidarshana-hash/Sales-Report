# Vrinda Store Annual Sales Report 2022

A Microsoft Excel-based sales data analysis project for **Vrinda Store**, covering 31,047 orders across India for the year 2022.

## 📊 Dataset Overview

* **Total Orders:** 31,047
* **Total Revenue:** ₹2,11,76,377
* **Time Period:** January – December 2022
* **Geography:** Pan-India (multiple states)
* **Channels:** Amazon, Myntra, Flipkart, Ajio, Nalli, Meesho, Others

### Raw Data Columns

`Order ID` · `Customer ID` · `Gender` · `Age` · `Age Category` · `Date` · `Month` · `Status` · `Channel` · `SKU` · `Category` · `Size` · `Qty` · `Currency` · `Amount` · `Ship City` · `Ship State` · `Postal Code` · `Country` · `B2B`

---

## 🛠️ Tools Used

* **Microsoft Excel** — Pivot Tables, Charts, Slicers, Dashboard

## 💡 Business Recommendations

* Target **adult women** — they drive the majority of revenue
* Focus marketing on **Amazon and Myntra** as primary channels
* Prioritize **Maharashtra, Karnataka, and UP** for regional campaigns
* Ramp up inventory in **Q1 (Jan–Mar)** to capitalize on peak demand
* Investigate return/cancellation causes to improve the 7.7% non-delivery rate
